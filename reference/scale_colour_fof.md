# Colour Scale for Factory of the Future

**\[deprecated\]** `scale_colour_fof()` has been renamed
[`scale_colour_nrc()`](https://amwu-nrc.github.io/reportabs/reference/scale_colour_nrc.md)

## Usage

``` r
scale_colour_fof(palette = "main", discrete = TRUE, reverse = FALSE)
```

## Arguments

- palette:

  Name of the colour palette (default "main")

- discrete:

  Whether the colours should be discrete or continuous (default TRUE)

- reverse:

  Whether the order of the colours should be reversed (default FALSE)

## Value

ggplot2 colour scale
