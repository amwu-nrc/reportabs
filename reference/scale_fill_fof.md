# Fill scale for Factory of the Future colours

**\[deprecated\]** `scale_fill_fof()` has been renamed
[`scale_fill_nrc()`](https://amwu-nrc.github.io/reportabs/reference/scale_fill_nrc.md)

## Usage

``` r
scale_fill_fof(palette = "main", discrete = TRUE, reverse = FALSE)
```

## Arguments

- palette:

  Name of the colour palette (default "main")

- discrete:

  Whether the colours should be discrete or continuous (default TRUE)

- reverse:

  Whether the order of the colours should be reversed (default FALSE)

## Value

ggplot2 fill scale
