# Manufacturing Alliance Definitions

A subset of ANZSIC2006 which captures the Manufacturing Alliance
definitions of manufacturing sub-sectors.

## Usage

``` r
manufacturing_alliance
```

## Format

### `manufacturing_alliance`

A data frame with 166 rows and 9 columns:

- anzsic_division_code:

  AZSIC 2006 Division Code

- anzsic_division:

  ANZSIC 2006 Division Name

- anzsic_subdivision_code:

  ANZSIC 2006 Subdivision Code

- anzsic_subdivision:

  ANZSIC 2006 Subdivision Name

- anzsic_group_code:

  ANZISC 2006 Group Code

- anzsic_group:

  ANZSIC 2006 Group Name

- anzsic_class_code:

  ANZSIC 2006 Class Code

- anzsic_class:

  ANZSIC 2006 Class Name

- sub_sector:

  Manufacturing Alliance Sub-Sector
