# Render AITI Monthly Labour Force Briefing in .pdf format

Render AITI Monthly Labour Force Briefing in .pdf format

## Usage

``` r
render_monthly_briefing(state = "South Australia", output_format)
```

## Arguments

- state:

  String. Name of state (in full). Defaults to South Australia.

- output_format:

  Format to render the document. One of "html" or "pdf"
