# List available colours

**\[deprecated\]** `fof_cols()` has been renamed
[`nrc_cols()`](https://amwu-nrc.github.io/reportabs/reference/nrc_cols.md)

## Usage

``` r
fof_cols(colour = NULL)
```

## Arguments

- colour:

  A character of the name of the colour. If NULL, returns all available
  colours.

## Value

A named character vector of the defined colours in the package.
